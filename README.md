## 🎯 Description

**Grid2D** est un outil éducatif conçu pour visualiser et expérimenter avec :

- 🟩 **Distance Manhattan** : Calcul de la distance entre deux points sur une grille
- 🎯 **Rayon de mouvement** : Visualisation des zones accessibles en fonction d'un rayon limité
- 🖱️ **Interactions en temps réel** : Cliquez sur n'importe quelle case pour explorer les possibilités de mouvement
- 📚 **Apprentissage algorithmique** : Comprenez comment les jeux, les stratégies et les systèmes de pathfinding fonctionnent

### Cas d'usage idéaux :

🎮 Développeurs en **game development** (AI, pathfinding)

## ✨ Fonctionnalités

| Fonctionnalité | Description |
|---|---|
| 🎨 **Grille Interactive 10x10** | Configurable facilement dans le code (taille, dimensions) |
| 📏 **Système de Distance Manhattan** | Calcul en temps réel de la distance entre positions |
| 🟢 **Visualisation Dynamique** | Les zones accessibles s'affichent en **vert**, les autres en blanc |
| 🎯 **Rayon de Mouvement Configurable** | Par défaut : 3 cases (modifiable) |
| 🖱️ **Interaction Mouse Click** | Cliquez sur une case pour déplacer le joueur |
| ⚡ **Architecture Modulaire** | Séparation claire entre `Grille`, `Player` et logique principale |
