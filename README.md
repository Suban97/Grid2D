# Grid2D

Simulateur de mouvement sur grille développé avec Pygame.

## Description

Grid2D est un simulateur permettant de visualiser et d'interagir avec un système de déplacement sur grille basé sur la distance Manhattan. Cet outil est utile pour comprendre les concepts de pathfinding et de calcul de distances dans un espace en grille.

## Fonctionnalités

- Grille interactive 10x10 configurable
- Système de déplacement limité par rayon (3 cases par défaut)
- Visualisation dynamique des zones accessibles (en vert)
- Calcul de distance Manhattan en temps réel
- Interface simple et intuitive
